🚀 NEXT: DEPLOY ON HEROKU WEBSITE

Follow these simple steps (detailed in heroku-github-deploy.md):

1. Go to https://dashboard.heroku.com/
2. Create new app
3. Connect to your GitHub repository
4. Add Heroku Postgres add-on
5. Set Config Vars:
   - SECRET_KEY (generate new one)
   - DEBUG=False
   - ALLOWED_HOSTS=your-app-name.herokuapp.com
6. Deploy from master branch
7. Run console: python manage.py createsuperuser
8. Open app!

Total time: ~5 minutes ⏱️
